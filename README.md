

## Installation procedure
1) git clone https://github.com/zhacked/todo_exam.git
2) cp .env.example .env
3) set up .env and database
4) php artisan migrate
5) npm install
6) composer install
7) php artisan passport:install
8) php artisan generate:key

## To start
1) php artisan serve
2) npm run watch
